# OAuthBM

`OAuthBM` is a `Vapor` package written in `Swift` that takes care of OAuth-2 interactions of your app with your provider.

### Getting Started

To get started with `OAuthBM`, take a look at [GettingStarted.md](/GettingStarted.md)
